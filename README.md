# 1: 全局安装  ts-node-dev    ts代码转js  
   ```
   yarn global add ts-node-dev typescript
   ```
# 2: 安装依赖 
   ```
   yarn add      or       npm i
   ```
# 3:开启服务
  ```
  yarn start
  ```

  ts 代码自动热更新
